# SistemaGerenciamentoBiblioteca

**Descrição do programa**
  Desenvolvimento de um programa para gerenciar um biblioteca, onde o sistema terá:
    Cadastro de Livros(título, autor, ano, disponibilidade)
    Cadastro de Usuários (nome, id unico, livros emprestados)
    Metodos de Emprestar e Deveolver livros
    Listar livros disponiveis, emprestados e autor
    Buscar livros por titulos ou autor

  **1 Estruturas dos dados**
    classe livro
      atributos: titulo, autor, ano, disponibilidade
    Classe Usuario
      atributos: nome, id, livros_emprestados
  
  **2 Funcionalidades Obrigatorios**
    Menu principal
      1. Cadastrar livro
      2. Cadastro usuario
      3. Emprestar livro
      4. Devolver livro
      5. Listar livros disponíveis
      6. Listar livros emprestados
      7. Buscar livro (título ou ISBN)
      8. Listar livros por autor
      9. Relatório estatístico
      0. Sair

  **3 Validações obrigatorios**
    ID unico
    Emprestimo de livro disponivel apenas se livro estiver disponivel E existir
    Limite de emprestimo de 3 livros por usuario
    Livro so pode ser emprestado para 1 úsuario por vez

  **4 Relatório Estatísticos**
    Total de livros cadastrados
    Total de livros disponíveis
    Total de usuários cadastrados
    Usuário com mais livros emprestados
    Autor com mais livros na biblioteca
